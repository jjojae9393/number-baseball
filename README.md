# 노리터 (Noritor)

> 어른들의 놀이터 — 2인 이상 실시간 매칭 종합 게임 플랫폼

---

## About

혼자가 아닌, 최소 2인 이상 매칭해서 즐기는 게임들로 이루어진 종합 놀이터입니다.

---

## Games

| # | 게임 | 인원 | 상태 |
|---|------|------|------|
| 1 | 숫자야구 | 2인 | 개발 완료 |
| 2 | 오목 | 2인 | 예정 |
| 3 | 마피아 게임 | 4인+ | 예정 |
| 4 | 끝말잇기 | 2인+ | 예정 |

## Roadmap

### Phase 1 — 인증 시스템
- [ ] 회원가입 / 로그인 / 로그아웃
- [ ] Firebase Authentication 연동

### Phase 2 — 프로필
- [ ] 내 정보 보기 및 수정 (닉네임, 프로필 등)

### Phase 3 — 전적 시스템
- [ ] 게임별 전적 기록 (승 / 패 / 승률)
- [ ] 프로필에서 전적 확인

### Phase 4 — 게임 허브
- [ ] 로그인 후 게임 선택 메뉴 (게임 로비)
- [ ] 각 게임별 독립 방 시스템

### Phase 5 — 게임 확장
- [ ] 오목
- [ ] 마피아 게임
- [ ] 끝말잇기

## Tech Stack

- **Frontend** — React (Vite)
- **Backend** — Firebase Realtime Database
- **Auth** — Firebase Authentication
- **Deploy** — GitHub Pages
- **Font** — Jua (Google Fonts)

## Getting Started

```bash
npm install
```
